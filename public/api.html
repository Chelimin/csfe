PropGenius APIs



  • Unit-Forecast API
    About: This API fetches the forecasted psf of a UNIT for a given quarter.

    • The query end point is used to process the input parameters: project name, floor, quarter of query, and area.

    • The query returns structured data in JSON format.
    Authentication: Please contact us for your API key.
    URL: http://api.catchsense.com/v1/unit-forecast
    Method: POST
    JSON Params: Required:
    projectName=[string]
    The Project Name for the unit(according to URA), in upper case.
    Example: projectName="THE BAYSHORE"

    floor=[integer]
    The floor / level the unit is at.
    Example: floor=5 

    quarter=[string]
    TThe quarter for the prediction.
    Example: quarter=”2016Q4” 

    typeOfSale=[string]
    Type of Sale for this unit. Possible values are ‘Resale’, ‘Sub Sale’, and ‘New Sale’
    Example: typeOfSale="Resale"

    area=[integer]
    The area of the unit.
    Example: area=1000

    propertyType=[string]
    The project type. Possible values are ‘Condominium, Apartment, and Executive Condominium’
    Example: projectType=”Condominium”

    token=[string]
    This is the given API key. Please do not lose or share it.
    Example: token =”[Your API Key Here]”
    Example: 
    curl -XPOST -d 
'{
"projectName": [string], 
"propertyType": [string], 
"floor": [integer],
"quarter": [string], 
"typeOfSale": [string], 
"area": [integer],
"token": [string] 
}'
    Copy and paste this snippet, but change the parameters accordingly:

    curl -i -XPOST -d '{"projectName": "ONE AMBER", "propertyType": "Condominium", "floor": 5,"quarter": "2016Q4", "area": 900, "typeOfSale": "Resale", "token": "[API key]"}' api.catchsense.com/v1/unit-forecast
    Success Response: Code: 200
    Content: 
    {
  "result": {
    "high": "123.456", 
    "low": "123.456", 
    "unitPred": "123.456"
  }
}

    Error Response: Code: 401 UNAUTHORIZED
    Content: 
    { "error" : "Access denied"}

    Notes: Last updated on 12/12/2016
